간단 설명서

아래의 프로젝트 설명을 참조해주세요.

  • 사용된 라이브러리:
    • 데이터 수집: dart_fss, pykrx
    • 데이터 가공: pandas
    • 웹서버 구축: flask
    • 표 꾸미기: datatables

  • 자료 기준일 관련:
    • 제품 키워드는 dart_fss 라이브러리(kind 공시 시스템 참조)를 참조하고 있습니다.
    • 시가총액 및 주식가치지표(멀티플)는 pykrx 라이브러리를 참조하고 있습니다.
    • 시장가격 데이터 베이스는 [2023년 4월 20일] 이후 데이터로만 구축되어 있습니다.
    • 데이터 업데이트 주기는 [매일 1회]입니다.

  • 검색 방법:
    • 메인 검색창에서 검색한 키워드는 제품 열에서 검색됩니다.
    • 정규식을 사용하여 검색 키워드를 수정할 수 있습니다. (예: 2차전지 또는 양극활물질 검색시 - "2차전지|활물질")
    • 검색 키워드를 공란으로 입력시 전체 상장사가 조회됩니다.
    • 표 우측 상단 검색창은 데이터 테이블 내 모든 영역을 대상으로 검색됩니다.
